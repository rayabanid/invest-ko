# InvestKo

InvestKo is a simple vuejs app to buy and sell stock.

### Build Setup

``` bash
# clone repo
git clone https://github.com/rayabanid/invest-ko.git

# cd directory
cd invest-ko

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
$ npm run build
```
